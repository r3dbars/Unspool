# Changelog

## v0.1.0 - 2026-05-04

First private release of Unspool.

### Added

- Minimal daily writing page
- `Unspool the noise` empty-state prompt
- Autosaved local Markdown entries
- Same-day session resume
- Fresh page on a new day
- 750-word daily completion moment
- Word count, streak, and simple stats
- Custom entries folder picker
- Light/dark mode
- Font size and font style controls
- Optional no-delete freewriting mode
- GitHub README, docs, app screenshot, icon, and DMG packaging

### Notes

- The DMG is intended for private testing.
- Public distribution still needs Developer ID signing and notarization.
- The writing experience was inspired by Farza's MIT-licensed Freewrite project.
