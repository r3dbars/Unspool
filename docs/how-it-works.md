# How Unspool Works

Unspool is intentionally small.

## The Daily Flow

1. The app opens to today's page.
2. The cursor is already in the editor.
3. The placeholder says `Unspool the noise`.
4. The placeholder disappears as soon as you write.
5. The page autosaves as Markdown.
6. At 750 words, the bottom bar shows a quiet completion moment.

If you reopen the app on the same day, Unspool opens the same page again.

If you open it on a new day, Unspool starts a fresh page.

## Why 750 Words

750 words is enough space to get past the first layer of thoughts.

The goal is not polished writing. The goal is to write long enough that the real pressure, repeated themes, and open loops start to show up.

## Local Markdown

Every page is saved as a Markdown file.

The default folder is:

```text
~/Library/Application Support/Unspool/Entries
```

You can choose a different folder in Settings.

That makes Unspool useful as an input for Obsidian, a local second brain, a Git-backed journal, or any agent workflow that can read Markdown.

## The Main Promise

Unspool should not turn writing into homework.

The app should stay quiet, private, local, and focused on the page.
