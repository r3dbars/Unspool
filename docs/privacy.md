# Privacy

Unspool is local first.

## What Stays Local

- daily pages
- word count and streak data
- selected entries folder
- theme and font preferences

By default, entries are saved here:

```text
~/Library/Application Support/Unspool/Entries
```

You can choose a different folder in Settings.

## What Unspool Does Not Do

- no account
- no cloud sync
- no analytics
- no social features
- no automatic AI calls
- no automatic upload of writing

## Markdown Files

Each entry is a plain Markdown file.

That is intentional. You can inspect it, back it up, move it, version it, or point another local tool at it.

## Experimental Code

The repo still contains experimental local reflection/export code from an earlier prototype.

That code is not part of the core daily writing flow. The product rule is still: nothing becomes outside context unless the user chooses it.
