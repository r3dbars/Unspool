# Unspool Assets

These files are used for the public repo and packaged macOS app.

- `Brand/unspool-app-icon-source.png`: source image for the app icon
- `AppIcon/Unspool.icns`: macOS icon copied into `Unspool.app`
- `GitHub/unspool-social-preview.png`: cover image for the README and GitHub social preview
- `GitHub/unspool-app-screenshot.png`: clean app screenshot used in the README

Regenerate `Unspool.icns` from the source image if the icon art changes.
